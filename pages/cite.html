Tag <cite>




Exemplos


Defina o título de uma obra com a tag <cite>:

<p><cite>The Scream</cite> by Edward Munch. Painted in 1893.</p>


Definição de Uso


A <cite>etiqueta define o título de uma obra criativa (por exemplo, um livro, um poema, uma música, um filme,
uma pintura, uma escultura, etc.).


Nota: O nome de uma pessoa não é o título de uma obra.


O texto no <cite> elemento geralmente é renderizado em itálico .
