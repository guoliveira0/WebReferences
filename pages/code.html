Tag <code>




Exemplos


Defina algum texto como código de computador em um documento:

<p>A tag HTML <code>button/<code> define um botão clicável</p>


<p>propriedade CSS <code>background-color</code> define a cor de fundo de um elemento.</p>


Definição de Uso


A tag <code>é usada para definir um pedaço de código de computador.
O conteúdo interno é exibido na fonte monoespaçada padrão do navegador.


Dica: esta tag não está obsoleta. No entanto, é possível obter um efeito mais rico usando CSS (veja o exemplo abaixo).
