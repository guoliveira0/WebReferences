Tag <col>




Exemplo


Defina a cor de fundo das três colunas com as tags <colgroup> e <col>:



<table>
  <colgroup>
   <col span="2" style="backgroud-color:red">
   <col style="backgroud-color:yellow">
  </colgroup>
  <tr>
   <th>ISBN</th>
   <th>Title</th>
   <th>Price</th>
  </tr>
  <tr>
   <td>3476896</td>
   <td>My first HTML</td>
   <td>$53</td>
  </tr>
</table>


Definição e Uso


A tag <col> especifica as propriedades da coluna para cada coluna dentro de um elemento <colgroup>.

A tag <col> é útil para aplicar estilos a colunas inteiras, em vez de repetir os estilos para cada célula, para cada linha.
