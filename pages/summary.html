Tag <summary>




Exemplos


Uma tabela com caption:

 <details>
  <summary>Epcot Center</summary>
  <p>Epcot is a theme park at Walt Disney World Resort featuring exciting attractions, international pavilions,
  award-winning fireworks and seasonal special events.</p>
 <details>



Definição de Uso


A tag <summary> define um cabeçalho visível para o elemento <details>.

O título pode ser clicado para visualizar/ocultar os detalhes.

Nota: O elemento <summary> deve ser o primeiro elemento filho do elemento <details>.
