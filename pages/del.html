Tag <del>




Exemplos


Um texto com uma parte excluída e uma nova parte inserida:


<p>My favorite color is <del>blue<del> <ins>red<ins>!<p>


Definição e Uso


A tag define o texto que foi excluído de um documento. Os navegadores geralmente marcam uma linha no texto excluído.

Dica: Observe também a tag <ins> para marcar o texto inserido.
