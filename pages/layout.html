Tag <h1>-<h6>




Exemplos


Existem seis tipos de titulos:

<h1> é o titulo nível 1 </h1>


<h2> é o titulo nível 2 </h2>


<h3> é o titulo nível 3 </h3>


<h4> é o titulo nível 4 </h4>


<h5> é o titulo nível 5 </h5>


<h6> é o titulo nível 6 </h6>


Definição de Uso


As tags <h1> to <h6> são usadas para definir cabeçalhos HTML. <h1> define o cabeçalho mais importante,
já o <h6> define o título menos importante. Nota: Use apenas um <h1> por página - isso deve representar o
cabeçalho/assunto principal de toda a página. Além disso, não pule os níveis de títulos - comece com <h1>,
depois use <h2>, e assim por diante.
