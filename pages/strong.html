Tag <strong>




Exemplos


Define algum texto importante em um documento:


<strong>Este texto é importante!</strong>


Definição e Uso


A tag <strong> é usada para definir um parte do texto que é importante para o documento.
O conteúdo dentro da tag é normalmente exibido em negrito.

Dica: Use a tag <b> para colocar textos em negrito, mas sem o valor semântico de importância!
