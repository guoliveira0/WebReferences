Tag <figure>




Exemplos


Use o elemento <figure> para marcar uma foto em um documento e o elemento <figcaption> para definir uma legenda para a foto:

<figure>

 <img src="pic_trulli.jpg" alt="Trulli" style="width:100%">

 <figcaption>Fig.1 - Trulli, Puglia, Italy.</figcaption>

<figure>


Definição e Uso


A tag <figure> especifica um conteúdo independente, como ilustrações, diagramas,
fotos, listangens de código, etc. Enquanto o conteúdo do elemento <figure> está relacionado
ao fluxo principal, sua posição é independente do fluxo principal e, se removido,
não deve afetar o fluxo do documento. Dica: O elemento <figcaption> é usado para adicionar
uma legenda para o elemento <figure>.
