Tag <output>




Exemplos


Execute um cálculo e mostre o resultado em um elemento <output>:

<form oninput="x.value=parseInt(a.value)+parseInt(b.value)">
 <input type="range" id="a" value="50">
 +<input type="number" id="b" value="25">
 =<output name="x" for="a b"></output>
</form>



Definição de Uso


A tag <output> é usada para representar o resultado de um cálculo(como um realizado por um script).


Suporte para Navegadores


Os números na tabela especificam a primeira versão do navegador que suporta toltalmente o elemento.


Navegadores
Compatibilidade 10.0 13.0 4.0 5.1 11.0


Atributo Valor Descrição
for element_id Especifica a relação entre o resultado do cálculo e os elementos usados ​​no cálculo
form form_id Especifica a qual formulário o elemento de saída pertence
name name Especifica um nome para o elemento de saída
