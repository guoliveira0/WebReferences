Tag <html>




Exemplo


Código HTML simples:


<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <title> Titulo do documento <title>
</head>
<body>
  <h1> Isso é um titulo </h1>
  <p> Isso é um parágrafo </p>
</body>
</html>


Definição e Uso


A tag <html> representa a raiz de um documento HTML.

A tag <html> é o recipiente para todos os outros elementos HTML, menos a tag <DOCTYPE>.

Nota: Você deve sempre incluir o atributo lang dentro da tag <html>, para declarar o idioma da página da Web.
Isso se destina a ajudar os motores de busca e navegadores.
