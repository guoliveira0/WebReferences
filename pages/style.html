Tag <style>




Exemplos


O uso da tag <style> aplica uma folha de estilo simples a um documento HTML:

<html>
<head>
<style>
  h1 {color:red;}
  p {color:blue;}
</style>
</head>
<body>

<h1>Um cabeçalho</h1>
<p>Um paragrafo</p>

</body>
</html>



Definição de Uso


A tag <style> é usada para definir informações de estilo (CSS) para um documento.

Dentro do elemento <style> você especifica como os elementos HTML devem ser renderizados em um navegador.



Dicas e Observação


Observação: quando um navegador lê uma folha de estilo, ele formata o documento HTML de acordo com as
informações da folha de estilo. Caso algumas propriedades tenham sido definidas para o mesmo seletor
(elemento) em diferentes folhas de estilo, será utilizado o valor da última folha de estilo lida.

Dica: Para vincular a uma folha de estilo externa, use a tag <link>.
