Tag <article>




Exemplos


Três artigos com conteúdo independente e independente:

<article>
 <h2>Google Chrome</h2>
 <p>Google Chrome é um navegador web desenvolvido pela Google, lançado em 2008. Chrome é o navegador da web mais popular do mundo hoje!</p>
</article>

<article>
 <h2>Mozilla Firefox</h2>
 <p>Mozilla Firefox é um navegador web de código aberto desenvolvido pela Mozilla. O Firefox é o segundo navegador mais popular desde janeiro,  2018.</p>
</article>

<article>
 <h2>Microsoft Edge</h2>
 <p>O Microsoft Edge é um navegador da Web desenvolvido pela Microsoft, lançado em 2015. O Microsoft Edge substituiu o Internet Explorer.</p>
</article>


Definição de Uso


A tag <article> especifica conteúdo independente e independente.

Um artigo deve fazer sentido por si só e deve ser possível distribuí-lo independentemente do resto do site.

Fontes potenciais para o elemento <article>:

  • Postagem em fórum
  • postagem no blog
  • Notícia

Observação: o elemento <article> não é exibido como algo especial em um navegador. No entanto,
você pode usar CSS para estilizar o elemento <article>.
