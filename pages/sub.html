Tag <sub>




Exemplos


Texto subscrito:

<p><This text contains <sub>subscript</sub> text.</p>


Definição de Uso


A <sub>tag define o texto subscrito.O texto subscrito aparece meio caractere abaixo da linha normal e,
às vezes, é renderizado em uma fonte menor. O texto subscrito pode ser usado para fórmulas químicas, como H 2 O.


Dica: Use a tag <sup> para definir o texto sobrescrito.
