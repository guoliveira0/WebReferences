Tag <s>




Exemplo

<s>Restam apenas 50 ingressos!</s></p>

<p>VENDIDOS!</p>


Definição de Uso


A tag <s> especifica o texto que não é mais correto, preciso ou relevante. O texto será exibido com uma linha através dele.
A tag <s> não deve ser usada para definir texto deletado em um documento, use a tag <del> para isso.
